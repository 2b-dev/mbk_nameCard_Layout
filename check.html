แนะนำเปิดเว็บนี้ด้วย Google Chrome

1. กดไอคอน Option

2. เลือก คัดลอกลิงก์

3. เปิดแอปพลิเคชัน Google Chrome

4. เข้าเว็บไซต์ที่คัดลอกลิงก์มา

Instrution open website with Google Chrome

1. Press the icon option

2. Select copy link

3. Open Google Chrome application

4. Paste copy link and enter the website
